Understanding the Key Principle

What is Recognition vs Recall?

RECALL means remembering information from memory without any help.

  • Example: Remembering that Ctrl+S saves a document
  • Problem: Users must be trained and remember commands
  • Difficult for new or occasional users

RECOGNITION means identifying something when you see it.

  • Example: Seeing a floppy disc icon and knowing it means "save"
  • Benefit: Users can understand immediately without training
  • Much easier for all users
Core HCI Principle: Good interface design uses visual icons and symbols (recognition) rather than requiring users to remember commands (recall). This makes systems easier to learn and use.
