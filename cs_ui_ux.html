üìù Annotated Tasks

  • Target User Analysis: Write a brief description of who you think each interface is designed for and whether it succeeds

üîç Detective Investigation Questions

For each interface, investigate these areas:

  • Basic Info: What is it? Who uses it? What's its purpose?
  • First Impressions: What do you notice first? How does it make you feel?
  • Ease of Use: Can you figure it out without help? Rate it 1-10.
  • Visual Design: Colours, text size, button clarity, layout logic
  • Accessibility: Could someone with vision problems use it easily?
  • Good vs Bad: What works well? What's confusing or annoying?
  • Improvements: What would you change as the designer?
